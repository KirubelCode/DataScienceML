Learn More: Salary Prediction Model

About the Project

The Salary Prediction Model is a machine learning-driven application designed to predict employee salaries based on input data such as economic sectors and optional parameters like company size or region. The model utilises Linear Regression for its predictions, offering businesses and HR professionals valuable insights for salary planning and budget allocation.

Features

  • Interactive CSV file upload, allowing users to input custom data for predictions.
  • Dynamic selection of columns for salaries, economic sectors, and optional parameters like company size or region.
  • Data cleaning and validation to handle missing or invalid values automatically.
  • Scatter plot visualisations of actual versus predicted salaries, colour-coded by economic sector.
  • Evaluation metrics, such as Mean Squared Error (MSE) and RÂ² Score, to assess model accuracy.

Technical Details

The model employs the following technologies:

  • Python: For data manipulation and machine learning.
  • Flask: To build the web-based interface.
  • Pandas and scikit-learn: For data processing and model training.
  • Matplotlib: For creating scatter plot visualisations.

How It Works

Users upload a CSV file containing their data. The application cleans the data, removing invalid or missing values, and dynamically adapts to user-selected columns for salary, economic sectors, and optional parameters. A linear regression model is then trained on this data to predict salaries. The results are visualised in a scatter plot, allowing users to compare actual versus predicted salaries.

Applications

  • Helps organisations forecast salary trends for better budget planning.
  • Assists HR professionals in analysing market standards and salary distributions.
  • Provides insights for decision-making in salary negotiations and workforce planning.

Limitations

While the model provides valuable insights, its accuracy depends on the quality and completeness of the input data. Predictions are limited to linear relationships between the selected features and salary outcomes.
