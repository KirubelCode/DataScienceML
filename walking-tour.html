Walking Tour Project

Overview

The Walking Tour Project is a Java-based tool designed to create and analyse graphs representing geographical sites. Users can import data via CSV files, where each site is represented as a node, and connections between sites are represented as edges with distances. This tool helps users analyse relationships between sites, such as finding the closest or farthest points, making it invaluable for route planning and decision-making.

Features

  • Create graph objects with nodes and edges from CSV file input.
  • Analyse node relationships, including finding closest and farthest sites.
  • Visualise connections using custom classes for nodes and edges.
  • Flexible methods to add nodes, edges, and calculate distances between sites.

Implementation

The project is implemented in Java, using object-oriented principles for easy extendability:

  • Node Class: Represents each site with attributes such as latitude, longitude, and adjacency lists to store connected nodes and distances.
  • Edge Class: Models connections between nodes with source, destination, and distance properties.
  • Graph Class: Manages a collection of nodes and edges, with methods to add new nodes, find closest sites, and analyse graph relationships.

How It Works

After importing a CSV file containing site coordinates, the program creates a graph representation:

  • Nodes are initialised with site details like name, latitude, and longitude.
  • Edges are created based on connections between nodes, with distances calculated dynamically.
  • Analysis methods find relationships like the closest and second closest nodes, helping users optimise their routes.

Applications

  • Route optimisation for walking tours or delivery services.
  • Geographical analysis for tourism or urban planning.
  • Distance-based decision-making.
