Learn More - Swimming Manager

Overview

The Swimming Manager is a Python-based application tailored for swimming coaches to better manage their swimmer data. Built with data import capabilities, it processes files containing swimmer information and organises them by age group and division. The application generates detailed graphs to aid in analysing swimmer performance, such as lap times and improvements over time.

Key Features

  • Import swimmer data directly from CSV or other file formats.
  • Categorise swimmers based on age, division, or performance metrics.
  • Generate visual graphs, such as line charts and bar graphs, to track swim times and progression.
  • User-friendly interface for data input and graph generation.

Technical Details

  • Developed in Python using Flask.
  • Focuses on seamless data manipulation and visualisation.
  • Designed for cloud deployment use for higher accessibility.
